## Getting Started

Before running the development server, make sure to install dependencies, prepare Husky, and setup environment variables:

```bash
# Install dependencies
npm install

# Prepare Husky hooks
npm run prepare

# Setup environment variables
cp .env.example .env
# Then edit .env as needed
```
